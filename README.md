# UnBlock v1.0.1 — macOS сборка через GitHub Actions

WebSocket-прокси для Telegram Desktop с современным GUI на PyQt6.

Этот репозиторий настроен так, что готовый macOS-приложение `.app` собирается
автоматически в облаке GitHub — **локально на Mac ничего ставить не нужно**.

---

## ⚡ Что делает этот репозиторий

При каждом пуше в ветку `main` (или при ручном запуске) GitHub Actions:

1. Запускает виртуальную машину macOS (Intel) в облаке
2. Ставит Python 3.11 и все зависимости
3. Собирает `UnBlock.app` через PyInstaller
4. Подписывает приложение ad-hoc подписью (убирает ошибку «повреждено»)
5. Упаковывает в `.zip` и публикует в двух местах:
   - **Artifacts** во вкладке Actions (хранение 90 дней)
   - **Releases** — постоянная ссылка на скачивание

---

## 📥 Как скачать готовый .app

### Способ 1: через Releases (рекомендуется)

1. В репозитории нажми вкладку **Releases** (справа в списке)
2. Найди последний релиз `UnBlock v1.0.1 (build #N)`
3. Скачай файл `UnBlock-macOS.zip` в разделе **Assets**

### Способ 2: через Actions

1. В репозитории нажми вкладку **Actions**
2. Выбери последний успешный запуск `Build macOS App`
3. Прокрути вниз до раздела **Artifacts**
4. Скачай `UnBlock-macOS-zip` или `UnBlock-macOS-app`

## 🚀 Установка на Mac

1. Распакуй `UnBlock-macOS.zip` двойным кликом
2. Перенеси `UnBlock.app` в папку **Программы**
3. **Первый запуск** (обязательно):
   - Правый клик на `UnBlock.app` → **Открыть**
   - В диалоге нажми **«Открыть»** (подтверждение)
4. Если пишет «приложение повреждено» — открой **Терминал** и выполни:
   ```bash
   xattr -cr /Applications/UnBlock.app
   ```
   После этого приложение запустится без проблем.

## 🔧 Настройка Telegram

1. Открой Telegram Desktop
2. **Настройки → Продвинутые → Тип соединения**
3. Выбери **«Использовать прокси»**
4. Тип: **SOCKS5**
5. Хост: `127.0.0.1`
6. Порт: `1080`
7. Нажми **«Сохранить»**

---

## 🛠 Ручной запуск сборки

Если хочешь пересобрать без изменения кода:

1. Открой вкладку **Actions**
2. Слева выбери **Build macOS App**
3. Справа нажми кнопку **Run workflow** → **Run workflow**
4. Через ~5 минут появится новый артефакт

## ❓ Решение проблем

| Проблема | Решение |
|----------|---------|
| «Приложение повреждено» | `xattr -cr /Applications/UnBlock.app` в Терминале |
| «Не удалось проверить разработчика» | Правый клик → Открыть → подтвердить |
| Прокси не работает | Проверь порт 1080 в настройках Telegram |
| Приложение не запускается | Открой Терминал: `/Applications/UnBlock.app/Contents/MacOS/UnBlock` — увидишь ошибки |
| На Apple Silicon (M1/M2/M3) медленно | Это нормально, работает через Rosetta 2. Скорость всё равно достаточная. |
